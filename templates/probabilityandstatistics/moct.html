Measures of Central Tendency (Mean, Mode and Median)

Mean

There are several types of mean, but we are going to discuss only three, which are: 

Arithmetic Mean
Geometric Mean
Harmonic Mean

Arithmetic Mean (AM)

This is the standard average.
It is normally given as:

AM = ∑fx/∑f

It is heavily used in insurance, economics, and data science.
AM helps identify:

  • Average claim amount
  • Average premium
  • Expected loss

Example

Given data values: 2, 4, 6, 8
AM = (2 + 4 + 6 + 8) / 4 = 20 / 4 = 5

Geometric Mean (GM)

Used when data grows multiplicatively (for example, population growth or investment returns).

Usually given as:

GM = n√(x1 · x2 · x3 · ... · xn)

Note:
AM ≥ GM
Best for compound interest and investment returns.

Example

Suppose returns are 10% and 20%:
GM = √(1.10 × 1.20) ≈ 1.1488 → 14.88% average growth rate.

Harmonic Mean (HM)

HM is used when averaging rates such as speed, density, and price per unit.

Usually given as:

HM = N / (∑(f/x))

Note:
HM ≤ GM ≤ AM
Usually used in actuarial risk rates and travel speed problems.

Example

A car travels 60 km/h and 40 km/h for equal distances.
HM = 2 / (1/60 + 1/40) = 48 km/h.

Median

The median is the middle value of a dataset when arranged in ascending or descending order.

1. Ungrouped Data

  • Arrange the data in ascending order.
  • Find the position of the median using: 
    Median position =
    if n is odd: (n + 1) / 2
    if n is even: n / 2 and (n / 2) + 1
    where n is the total number of observations.
  • Locate the median using the position from the formula.

Example:

Data: 3, 7, 8, 5, 12, 14, 21

Step 1: Arrange: 3, 5, 7, 8, 12, 14, 21

Step 2: n = 7 (odd) → Median position = (7+1)/2 = 4

Step 3: Median = 4th value = 8

2. Grouped Data

For grouped data, the median is calculated as:

Median = L + ((n/2 - CF)/f) * h
  • L = lower boundary of the median class
  • n = total frequency
  • CF = cumulative frequency before the median class
  • f = frequency of the median class
  • h = class width

Example:

ClassFrequency
0-105
10-208
20-3012
30-405

Step 1: Total frequency n = 30 → n/2 = 15

Step 2: Median class = 20-30 (cumulative frequency before = 13)

Step 3: Median = 20 + ((15-13)/12) * 10 = 21.67

Mode

The mode is the most frequently occurring value.

1. Ungrouped Data

Example: Data: 4, 6, 4, 9, 10, 4, 6 → Mode = 4

2. Grouped Data

Formula for grouped data:

Mode = L + ((f1 - f0) / (2f1 - f0 - f2)) * h
  • L = lower boundary of modal class
  • f1 = frequency of modal class
  • f0 = frequency before modal class
  • f2 = frequency after modal class
  • h = class width

Example:

ClassFrequency
0-105
10-2012
20-308
30-405

Step 1: Modal class = 10-20 (f1 = 12, f0 = 5, f2 = 8)

Step 2: Mode = 10 + ((12-5) / (2*12 - 5 - 8)) * 10 ≈ 16.36

Quartiles

Quartiles divide data into four equal parts:

  • Q1: 25% of data lies below
  • Q2 (Median): 50% of data lies below
  • Q3: 75% of data lies below

Ungrouped Data

Formula:

Q1 = (n+1)/4 th value
Q2 = Median
Q3 = 3(n+1)/4 th value

Example:

Data: 6, 7, 15, 36, 39, 40, 41, 42, 43, 47, 49

Q1 = 3rd value = 15, Q2 = Median = 40, Q3 = 9th value = 43

Grouped Data

Formula:

Qk = L + ((k*n/4 - CF)/f) * h
  • k = 1, 2, 3 for Q1, Q2, Q3 respectively
  • Other symbols as in median formula
