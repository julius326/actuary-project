Probability

Sample Space

Sample space (S) is the set of all possible outcomes of an experiment.

Example:
If two babies are selected and the sex of each baby is observed: 

S = {MM, FF, MF, FM}

Sample space may be:

  • Finite – limited outcomes (e.g., dice)
  • Infinite – unlimited outcomes (e.g., measuring height)

A sample space is:

  • Discrete if outcomes are countable (1,2,3,...)
  • Continuous if outcomes lie in intervals (5 < x < 9)

Event

An event E is any subset of the sample space including the empty set and S itself. 

If E contains m outcomes and total outcomes are n:

P(E) = m / n

0 ≤ P(E) ≤ 1

Example

Out of 751 graduates, 383 were not employed in their major. Probability that a graduate works outside their major: 

P = 383 / 751 = 0.51

Conditional Probability

Conditional probability of A given B: 

P(A | B) = P(A ∩ B) / P(B)

Example (Completed)

Given:

  • P(W) = 0.4
  • P(H) = 0.1
  • P(H ∩ W) = 0.08

(1) Probability wife attends given husband attends:

P(W | H) = P(W ∩ H) / P(H)
= 0.08 / 0.1
= 0.8

(2) Probability both attend:

P(W ∩ H) = 0.08

Random Variables

A random variable (X) assigns numerical values to outcomes.

1. Discrete Random Variable

Takes countable values.

Mean:     E(X) = Σ x P(x)
Variance: Var(X) = Σ (x - μ)² P(x)

2. Continuous Random Variable

Takes values over intervals.

Mean:     E(X) = ∫ x f(x) dx
Variance: Var(X) = ∫ (x - μ)² f(x) dx

Bernoulli Distribution

X ~ Bernoulli(p) 

P(X = x) = p^x (1 - p)^(1 - x)
Where x = 0 or 1

Mean = p
Variance = p(1 - p)

Example

5 good oranges, 15 bad. Total = 20 

p = 5/20 = 1/4

Mean = 1/4
Variance = (1/4)(3/4) = 3/16

Binomial Distribution

If X is number of successes in n independent Bernoulli trials: 

X ~ Bin(n, p)

P(X = x) = C(n, x) p^x (1 - p)^(n - x)

Mean = np
Variance = np(1 - p)

Example

A coin is tossed 5 times. Find probability of exactly 3 heads. 

n = 5, p = 1/2

P(X=3) = C(5,3)(1/2)^3(1/2)^2
= 10 * (1/32)
= 10/32
= 0.3125

Poisson Distribution

Used for rare events in fixed interval. 

X ~ Poisson(λ)

P(X = x) = (e^-λ λ^x) / x!

Mean = λ
Variance = λ

Example

If average accidents per day = 2, find probability of 3 accidents. 

λ = 2

P(X=3) = e^-2 (2^3) / 3!
= (0.1353 * 8) / 6
= 1.0824 / 6
= 0.1804

Normal Distribution

X ~ N(μ, σ²) 

f(x) = (1 / √(2πσ²)) e^(-(x - μ)² / 2σ²)

Mean = μ
Variance = σ²

Standard Normal

Z = (X - μ) / σ

Example

If μ = 100, σ = 10, find probability X < 110. 

Z = (110 - 100) / 10
= 1

From Z-table:
P(Z < 1) = 0.8413

Important Notes

  • Capital letters denote random variables (X, Y).
  • Lowercase letters denote specific values (x, y).
  • 0 ≤ P(A) ≤ 1
  • Total probability of sample space = 1
  • Var(X) = E(X²) − [E(X)]²
