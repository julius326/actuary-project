Measures of Dispersion

Measures of dispersion show how spread out the data is around a central value.

1. Range

The range is the difference between the largest and smallest value in a dataset.

Formula:

Range = Maximum value - Minimum value

Example: Data: 3, 7, 8, 5, 12 → Range = 12 - 3 = 9

2. Mean Deviation (MD)

Mean deviation measures the average distance of each observation from the mean or median.

Formula about mean:

MD = Σ |x_i - x̄| / n
  • x_i = each value
  • x̄ = mean
  • n = total number of observations

Example: Data: 2, 4, 6

Mean = (2+4+6)/3 = 4

MD = (|2-4| + |4-4| + |6-4|)/3 = (2+0+2)/3 = 1.33

3. Variance

Variance measures the average of squared deviations from the mean.

Formula (population):

σ² = Σ (x_i - μ)² / N

Formula (sample):

s² = Σ (x_i - x̄)² / (n-1)
  • μ = population mean
  • x̄ = sample mean
  • N = population size
  • n = sample size

Example: Data: 2, 4, 6 → Mean = 4

Variance (sample) = ((2-4)² + (4-4)² + (6-4)²)/(3-1) = (4+0+4)/2 = 4

4. Standard Deviation (SD)

The standard deviation is the square root of variance, showing dispersion in the same units as data.

Formula:

SD = √Variance

Example: Continuing the previous example → SD = √4 = 2

5. Quartile Deviation / Semi-Interquartile Range

Measures the spread of the middle 50% of the data.

Formula:

QD = (Q3 - Q1) / 2

Example: If Q1 = 15 and Q3 = 43 → QD = (43 - 15)/2 = 14
