BASIC MATHEMATICS

RELATIONS AND FUNCTIONS

Cartesian Products

Definition

Let A and B be any two sets. Then the Cartesian product of A and B, denoted by A × B, is defined as:
A × B = { (a, b) | a ∈ A and b ∈ B }
For example:
If A = {1, 2, a} and B = {x, y, z},
then A × B = { (1, x), (1, y), (1, z), (2, x), (2, y), (2, z), (a, x), (a, y), (a, z) }
RELATIONS

Definition:

Let A and B be any two non-empty sets. Then every non-empty subset of A × B is called a relation from A to B. In other words, a relation is a set of ordered pairs where the first element comes from A and the second from B.
Examples:
1. Let A = {1, 2, 3} and B = {x, y}. Then one possible relation R from A to B is:
R = { (1, x), (2, y) }
Another possible relation is the full Cartesian product:
R = { (1, x), (1, y), (2, x), (2, y), (3, x), (3, y) }

2. A relation on a single set (A = B) is often called a relation on A. For example:
Let A = {1, 2, 3}. Then a relation R on A could be:
R = { (1, 2), (2, 3), (3, 1) }

3. Relations can also describe real-life connections. For example:
Let A = {Students} = {Alice, Bob} and B = {Courses} = {Math, English}. Then the relation "is enrolled in" could be:
R = { (Alice, Math), (Bob, English), (Alice, English) }
Notes:
- Relations can be one-to-one, one-to-many, many-to-one, or many-to-many depending on how elements of A relate to elements of B.
- The total set of all possible relations from A to B is the power set of A × B.
EQUIVALENT RELATIONS
Definition:
Let S be a set. A relation E, which is a subset of S × S, is called an equivalence relation on S if it satisfies the following three properties:
  1. Reflexive: (a, a) ∈ E for every a ∈ S.
    Symbolically: ∀ a ∈ S, (a, a) ∈ E
  2. Symmetric: If (a, b) ∈ E, then (b, a) ∈ E for all a, b ∈ S.
    Symbolically: ∀ a, b ∈ S, (a, b) ∈ E ⇒ (b, a) ∈ E
  3. Transitive: If (a, b) ∈ E and (b, c) ∈ E, then (a, c) ∈ E for all a, b, c ∈ S.
    Symbolically: ∀ a, b, c ∈ S, (a, b) ∈ E ∧ (b, c) ∈ E ⇒ (a, c) ∈ E
Notes:
- “∀” is the symbol for for every or for all. - An equivalence relation partitions the set S into disjoint equivalence classes.
Example:
Let S = {1, 2, 3, 4} and define a relation E by a ~ b if a and b have the same parity (both even or both odd). Then:
  • E = {(1,1), (1,3), (3,1), (3,3), (2,2), (2,4), (4,2), (4,4)}
  • This relation is reflexive, symmetric, and transitive, so it is an equivalence relation.
PRODUCT OF FUNCTIONS
Definition:

Let f, g : ℝ → ℝ be two functions. The product of f and g is denoted by f · g (read as f dot g) and defined as:
(f · g)(x) = f(x) · g(x), ∀ x ∈ ℝ
Example:
If f(x) = 2x + 3 and g(x) = x² - 4, then:
(f · g)(x) = (2x + 3)(x² - 4) = 2x³ - 8x + 3x² - 12 = 2x³ + 3x² - 8x - 12
Remark: The product of functions is commutative, i.e., f · g = g · f.
COMPOSITION OF FUNCTIONS
Definition:

Let f, g : ℝ → ℝ be two functions. The composition of f and g is denoted by f ∘ g (read as f circle g) and defined as:
(f ∘ g)(x) = f(g(x)), ∀ x ∈ ℝ
Example:
Let f(x) = 5x + 1 and g(x) = 2x + 4. Then:
(f ∘ g)(x) = f(g(x)) = f(2x + 4) = 5(2x + 4) + 1 = 10x + 21
(g ∘ f)(x) = g(f(x)) = g(5x + 1) = 2(5x + 1) + 4 = 10x + 6
Remarks:
  • f · g = g · f → multiplication of functions is commutative.
  • f ∘ g ≠ g ∘ f → composition of functions is not commutative in general.
INVERSE OF A FUNCTION

To find the inverse of a function f(x):
  1. Write y = f(x)
  2. Swap x and y (make x the subject): solve for y in terms of x
  3. Replace y with f⁻¹(x)
Example:
f(x) = 2x + 3
y = 2x + 3 → solve for x: x = (y - 3)/2
Therefore, f⁻¹(x) = (x - 3)/2 = ½(x - 3)
Remark:
A function f(x) is not defined at points where it is impossible to compute its value, such as:
  • Division by zero
  • Square root of a negative number (in ℝ)
Example: Let f: ℝ → ℝ, f(x) = 1/(x - 1). Then:
- Domain: x ≠ 1 → D(f) = ℝ \ {1}
- Range: R(f) = ℝ \ {0}
ADDITIONAL IMPORTANT CONCEPTS
  • Domain of a function: Set of all x-values for which f(x) is defined.
  • Range of a function: Set of all possible output values f(x).
  • Identity function: f(x) = x. Its inverse is itself.
  • Zero function: f(x) = 0 for all x ∈ ℝ.
  • Composite functions and inverses: (f ∘ f⁻¹)(x) = x and (f⁻¹ ∘ f)(x) = x.
  • Important remark: Not all functions have inverses. A function must be one-to-one (injective) and onto (surjective) to have an inverse.

COMPLEX NUMBERS

Observe that the set ℝ of real numbers has no solution to the equation x2 + a = 0, ∀ a ∈ ℝ, when a < 0, since the square root of a negative number does not exist in ℝ. Therefore, there is a need to extend the real number system to include complex numbers.

Complex numbers are denoted by ℂ and defined as:

ℂ = { z : z = x + iy ; x, y ∈ ℝ and i2 = −1 }

REMARKS
  • i is called the imaginary unit
  • x is called the real part of z and is denoted by Re(z)
  • y is called the imaginary part of z and is denoted by Im(z)
  • If y = 0, then z is a real number
  • If x = 0, then z is a purely imaginary number
Basic Arithmetic of Complex Numbers

Let z1 and z2 be complex numbers such that z1 = a + ib, z2 = c + id, where a, b, c, d ∈ ℝ. 

i. Addition
   z₁ + z₂ = (a + ib) + (c + id)
           = (a + c) + i(b + d) ∈ ℂ
   Hence ℂ is closed under addition.

ii. Subtraction
   z₁ − z₂ = (a + ib) − (c + id)
           = (a − c) + i(b − d) ∈ ℂ
   Hence ℂ is closed under subtraction.

iii. Multiplication
   z₁ · z₂ = (a + ib)(c + id)
           = ac + iad + ibc + i²bd
           = ac − bd + i(ad + bc) ∈ ℂ
   Hence ℂ is closed under multiplication.
Example

Let z1 = −4 + 6i and z2 = 3 − 5i. 
z₁ + z₂ = (−4 + 6i) + (3 − 5i)
        = −1 + i

z₁ − z₂ = (−4 + 6i) − (3 − 5i)
        = −7 + 11i

z₁ · z₂ = (−4 + 6i)(3 − 5i)
        = −12 + 20i + 18i − 30i²
        = −12 + 38i + 30
        = 18 + 38i
Complex Conjugate

The complex conjugate of z = x + iy is defined as:

z̄ = x − iy

Example
If z = 3 + 4i, then z̄ = 3 − 4i
Division of Complex Numbers

To divide complex numbers, multiply the numerator and denominator by the conjugate of the denominator.

Example 
(2 + 3i) / (1 − i)

Multiply numerator and denominator by (1 + i):

= (2 + 3i)(1 + i) / (1 − i)(1 + i)
= (2 + 2i + 3i + 3i²) / (1 − i²)
= (2 + 5i − 3) / (1 + 1)
= (−1 + 5i) / 2
IMPORTANT REMARKS
  • ∀ x ∈ ℝ, x = x + 0i, hence ℝ ⊂ ℂ
  • Complex numbers cannot be ordered like real numbers
  • ℂ is closed under addition, subtraction, multiplication, and division (except by zero)
ARGAND DIAGRAM

An Argand diagram is a graphical representation of complex numbers. The horizontal axis represents the real part and the vertical axis represents the imaginary part.

Example: Plot the complex number z = 3 + 2i

  • Each complex number z = x + iy is represented by a point (x, y)
  • The real part is measured along the horizontal axis
  • The imaginary part is measured along the vertical axis
  • The distance from the origin is the modulus |z|
SQUARE ROOT OF COMPLEX NUMBERS

The square roots of a complex number z are the complex numbers w such that w² = z. Every non-zero complex number has exactly two square roots.

Let z = a + ib. Suppose √z = x + iy, where x, y ∈ ℝ. Then:

(x + iy)² = a + ib ⇒ x² − y² + 2ixy = a + ib

Equating real and imaginary parts:
x² − y² = a    (1)
2xy = b    (2)

Examples

i) Find √(2i)

Solution

Let √(2i) = x + iy
Then:
(x + iy)² = 2i
⇒ x² − y² + 2ixy = 0 + 2i

Equating real and imaginary parts:
x² − y² = 0    (1)
2xy = 2    (2)

From (2): xy = 1
From (1): x² = y² ⇒ x = y

Hence:
x² = 1 ⇒ x = ±1
y = x

Therefore:
√(2i) = ±(1 + i)

ii) Find √(5 − 12i)

Solution

Let √(5 − 12i) = x + iy
Then:
(x + iy)² = 5 − 12i
⇒ x² − y² + 2ixy = 5 − 12i

Equating real and imaginary parts:
x² − y² = 5    (1)
2xy = −12    (2)

From (2): xy = −6

Try x = 3, y = −2:
x² − y² = 9 − 4 = 5 ✓
xy = −6 ✓

Hence:
√(5 − 12i) = ±(3 − 2i)

Important Remarks
  • Every non-zero complex number has exactly two square roots.
  • If w is a square root of z, then −w is also a square root of z.
  • Square roots of complex numbers are generally not unique.
  • Solving by equating real and imaginary parts is the standard method.
COMPLEX ROOTS OF QUADRATIC EQUATIONS

A quadratic equation of the form
ax2 + bx + c = 0,   a ≠ 0
may have complex roots when its discriminant is negative.

The discriminant is given by:
Δ = b2 − 4ac

If Δ < 0, then the roots of the equation are complex conjugates.

Example

Solve:
x2 + 1 = 0

Solution

Comparing with ax2 + bx + c = 0:
a = 1,   b = 0,   c = 1

Discriminant:
Δ = b2 − 4ac
Δ = 0 − 4(1)(1)
Δ = −4 < 0

Hence, the roots are complex.

Using the quadratic formula:
x = −b ± √(b2 − 4ac) / 2a

x = ± √(−4) / 2
x = ± 2i / 2
x = ± i

Therefore, the complex roots are:
x = i and x = −i

Important Remarks
  • Quadratic equations with negative discriminant have no real roots.
  • Such equations have two complex conjugate roots.
  • If a + bi is a root, then a − bi is also a root.
  • The roots lie symmetrically about the real axis on the Argand diagram.

PERMUTATION AND COMBINATION

PERMUTATION

Assume we have three items A, B, and C and we want to arrange them in a row.
How many different ways can we arrange them?

A permutation is the total number of different ways of arranging objects where order matters.

Solution
3 2 1

Total arrangements = 3 × 2 × 1 = 6

Definition:
A permutation of a given number of objects is an arrangement of these objects in a definite order.

FACTORIAL NOTATION

The product of n consecutive natural numbers from 1 to n is denoted by n! and is called factorial n.

n! = n(n − 1)(n − 2) … 3 × 2 × 1

Examples:
5! = 5 × 4 × 3 × 2 × 1 = 120
0! = 1

PERMUTATION OF n OBJECTS TAKEN r AT A TIME (r ≤ n)

The permutation of n objects taken r at a time is denoted by nPr and is defined as:

nPr = n! / (n − r)!

Example:

In how many ways can 8 books be arranged on 4 shelves?

Solution
8 7 6 5 
n = 8
r = 4

nPr = 8P4 = 8! / (8 − 4)!
     = 8! / 4!
     = (8 × 7 × 6 × 5)
     = 1680
PERMUTATION OF n OBJECTS NOT ALL DISTINCT

If n objects consist of n₁ identical objects of one kind, n₂ identical objects of a second kind, …, nk identical objects of the kth kind, then the number of distinct permutations is: 

n! / (n₁! n₂! … nk!)
WORKED EXAMPLES

i) DEFEAT

Total letters = 6
E appears twice 

Number of arrangements = 6! / 2!
                        = 360

ii) ARRANGE

Total letters = 7
A appears twice
R appears twice 

Number of arrangements = 7! / (2! × 2!)
                        = 1260

iii) KISII

Total letters = 5
I appears three times 

Number of arrangements = 5! / 3!
                        = 20

iv) PERMUTATION

Total letters = 11
T appears twice 

Number of arrangements = 11! / 2!
                        = 19,958,400
IMPORTANT REMARKS
  • Order matters in permutations.
  • Factorials grow very fast.
  • Repeated objects reduce the total number of permutations.
  • nPr is not equal to nCr.
PERMUTATION OF OBJECTS WITH RESTRICTIONS

In some permutation problems, certain conditions (restrictions) must be satisfied. These conditions limit the total number of possible arrangements.

Example 1

How many numbers greater than 2000 can be formed from the digits 1, 2, 3, and 4 if each digit is used only once?

i) Even numbers greater than 2000

Solution

For the number to be:

  • Greater than 2000 ⇒ the first digit must be 2, 3, or 4
  • Even ⇒ the last digit must be 2 or 4

Possible cases:

Case 1: Last digit = 2
Remaining digits = {1, 3, 4}
First digit cannot be 1 (number would be < 2000)
Choices for first digit = 3 or 4 (2 ways)
Remaining two digits can be arranged in 2! ways

Total = 2 × 2! = 4

Case 2: Last digit = 4
Remaining digits = {1, 2, 3}
First digit cannot be 1
Choices for first digit = 2 or 3 (2 ways)
Remaining two digits can be arranged in 2! ways

Total = 2 × 2! = 4

Total even numbers greater than 2000 = 4 + 4 = 8

ii) Odd numbers greater than 2000

Solution

For an odd number, the last digit must be 1 or 3.

Case 1: Last digit = 1
Remaining digits = {2, 3, 4}
First digit must be 2, 3, or 4 (3 ways)
Remaining two digits arranged in 2! ways

Total = 3 × 2! = 6

Case 2: Last digit = 3
Remaining digits = {1, 2, 4}
First digit cannot be 1
Choices = 2 or 4 (2 ways)
Remaining digits in 2! ways

Total = 2 × 2! = 4

Total odd numbers greater than 2000 = 6 + 4 = 10

iii) Numbers greater than 3000

Solution

First digit must be 3 or 4 (2 ways).
Remaining three digits can be arranged in 3! ways.

Total = 2 × 3! = 12

Example 2

How many permutations can be formed from the word ADDING:
(a) when the two D's are together
(b) when the two D's are not together

Total letters in ADDING = 6
D appears twice

(a) When the two D's are together

Solution

Treat the two D's as one unit (DD).
Objects: DD, A, I, N, G → 5 objects

Number of permutations: 

5! = 120

Total arrangements = 120

(b) When the two D's are not together

Solution

First find total permutations without restriction: 

Total = 6! / 2! = 360

Subtract arrangements where D's are together: 

Required = 360 − 120 = 240

Total arrangements = 240

IMPORTANT REMARKS
  • Restrictions reduce the total number of permutations.
  • “Together” problems use the block (grouping) method.
  • “Not together” = total arrangements − together arrangements.
  • Always check digit conditions (even, odd, size).
COMBINATIONS

Assume we have 5 paintings available and only 3 of them can be displayed.
In how many different ways can we select the 3 paintings if the order does not matter?

A combination is a selection of objects where the order of selection does not matter.

If we have n objects from which we select r objects (where r < n), then the number of possible combinations is denoted by nCr and defined as:

nCr = nPr / r! = n! / (n − r)! r!

Note: Every combination is a permutation, but not every permutation is a combination.

Solution 
n = 5
r = 3

5C3 = 5P3 / 3!
     = (5! / (5 − 3)!) / 3!
     = 5! / (2! 3!)
     = (5 × 4 × 3 × 2 × 1) / (2 × 1 × 3 × 2 × 1)
     = 10

Therefore, the number of ways of selecting the 3 paintings is 10.

Additional Examples

Example 2:
In how many ways can a committee of 4 people be chosen from a group of 10 people? 

10C4 = 10! / (6! 4!)
     = 210

Example 3:
How many ways can 2 cards be chosen from a pack of 52 cards? 

52C2 = 52! / (50! 2!)
     = 1326
Important Remarks
  • Order does not matter in combinations.
  • nCr = nC(n − r).
  • nC0 = nCn = 1.
  • nCr is always a whole number.
